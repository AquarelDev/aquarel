Aquarel Aquarel est un site vitrine conçu pour être géré par des administrateurs avec des droits illimités, tandis que les adhérents disposent de droits limités. Ce projet est basé sur WordPress et est destiné à être déployé localement via WampServer. Prérequis Pour exécuter ce projet, vous aurez besoin des éléments suivants : WampServer Git WordPress Installation Étape 1 : Installer WampServer Téléchargez et installez WampServer à partir du site officiel. Suivez les instructions fournies pour compléter l'installation. Étape 2 : Cloner le dépôt Git Clonez le dépôt Git de votre projet Aquarel. Ouvrez une fenêtre de terminal et exécutez la commande suivante : Copier le code git clone Remplacez par l'URL de votre dépôt Git. Étape 3 : Déplacer le dossier du projet Déplacez le dossier cloné dans le répertoire www de WampServer. Par défaut, ce répertoire se trouve généralement à l'emplacement suivant : makefile Copier le code C:\wamp64\www\ Étape 4 : Lancer WampServer Ouvrez WampServer et démarrez les services Apache et MySQL. Vous pouvez accéder à l'interface de WampServer en cliquant sur l'icône WampServer dans la barre des tâches et en sélectionnant "Localhost" ou en cliquant sur ce lien. Étape 5 : Configurer WordPress Accédez à http://localhost/aquarel dans votre navigateur web pour commencer la configuration de WordPress. Suivez les instructions à l'écran pour configurer la base de données et finaliser l'installation de WordPress. Utilisation Une fois l'installation terminée, vous pouvez accéder à l'interface d'administration de WordPress via http://localhost/aquarel/wp-admin. Utilisez vos identifiants d'administrateur pour vous connecter et gérer le site. Les administrateurs disposent de droits illimités pour gérer le site, tandis que les adhérents ont des droits limités pour interagir avec le contenu selon les permissions définies.
